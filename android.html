Corona Virus Tracker Android App

If You're an Android user, and would like to be updated to the latest news about the virus outbreak, coronatrackers.com provides you that option with an application optimized for small screens like smartphones, additionally, it includes push notifications if enabled.

The app provides the same functionality as the site, and any feature added here, it will be updated in the smartphone app as well.


App features

Timeline

Semi real-time timeline updates about the cases, their locations, time (GMT), and date.

Cases Reports

Real-time reports on the latest cases reported around the world, suspected cases, and fatalities (as soon as the numbers are confirmed).

3D Map

3D map of the cases.

Per country statistics.

Per country statistics for confirmed cases and deaths.

Travel notices and restrictions.

Travel notices from the CDC.gov

Dashboards

Cases Graphs, historical trends.

PHEIC

PHEIC (Public Health Emergency of International Concern) index.

Tweets aggregator

Coronavirus Tweets aggregator.

Push notifications on the latest cases

Get notified once a day.
